# AI Email Digest Agent
An autonomous agent designed to solve information overload by intelligently summarizing incoming communications.

## 🛠 Tech Stack
* **Orchestration:** Make.com
* **Intelligence:** Anthropic Claude (via API)
* **Data Handling:** JSON Parsing & Advanced Webhooks

## 🧠 The Logic
This isn't a simple automation; it's an **agentic workflow**. 
1. **Context Extraction:** The agent monitors Gmail and extracts full email threads.
2. **Linguistic Analysis:** Utilizing prompt engineering to categorize emails by priority and sentiment.
3. **Structured Output:** Claude parses the data into a clean JSON format for the final delivery.

## 📁 How to Use
The included `.json` file is a Make.com Blueprint. You can import this into your own Make.com dashboard to replicate the architecture.
